# npm init

Az `npm init` paranccsal egy új NodeJS csomagot hozhatunk létre. A parancs kérdéseket tesz fel, majd létrehozza a `package.json` fájlt.

```
$ npm init
package name: (valami)
version: (1.0.0)
description:
entry point: (index.js)
test command:
git repository:
keywords:
author:
license: (ISC)
...
```
